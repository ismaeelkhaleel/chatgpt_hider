# ChatGPT Input Hider Extension

A simple Chromium-based browser extension (Chrome / Brave) that **hides the text you type into ChatGPT** and allows you to **toggle visibility from the extension popup**.

- Input box always stays visible
- Typed prompt can be **hidden or shown**
- Default state: **Input Hidden**
- Works on **chat.openai.com**
- No Node.js required
- No backend
- No payment needed for local use

Repository:  
https://github.com/ismaeelkhaleel/chatgpt_hider

---

## Features

- Masks ChatGPT input text using browser-native CSS
- Toggle visibility from popup
- Instant hide/show (no delay)
- Survives page reloads
- Works on Chrome & Brave (Chromium-based browsers)

## How to Use

1. Open **Open ChatGPT**
2. Start typing in the input box  
 ‚Üí text is **hidden by default**
3. Click the **extension icon**
4. Popup appears with a button:
 - **Close Eye** ‚Üí click to make text visible
 - **Open Eye** ‚Üí click to hide text again
5. Toggle anytime

---

## Default Behavior

- First load ‚Üí **Prompt Hide**
- Popup button always reflects current state
- Input box never disappears
- Only typed characters are masked/unmasked
